## Feature Reference

When adding screenshots, describe the feature each shows:

- **Catchment demographics** — Population, households, income bands by zone
- **Competitor mapping** — POI density, brand mix, saturation heat
- **Footfall / trade signals** — Daily patterns, peak hours, weekend boost
- **Micro-zone heat** — Radial or choropleth zone visualization
- **Site scoring** — Composite scores, viability, rent benchmarks
- **Retail mix** — Branded vs non-branded by category
